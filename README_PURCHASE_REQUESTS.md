# درخواست تماس مشتری از Checkout

## راه‌اندازی

1. فایل زیر را در Supabase SQL Editor اجرا کنید:

   `supabase/migrations/20260601_purchase_requests.sql`

2. پروژه را اجرا کنید:

   `pnpm install`
   `pnpm dev`

## مسیرها

- فرم مشتری: `/checkout`
- لیست درخواست‌ها: `/admin/purchase-requests`
- جزئیات درخواست: `/admin/purchase-requests/[id]`

## نکات امنیتی

- شماره تماس مشتری از طریق جدول‌ها به anon نمایش داده نمی‌شود.
- کلاینت عمومی مستقیم در جدول‌ها insert نمی‌کند؛ server action از RPC کنترل‌شده `create_purchase_request` استفاده می‌کند.
- RPC قیمت کالاها را از دیتابیس مجدداً محاسبه می‌کند و قیمت browser را قبول نمی‌کند.
- دسترسی جدول‌های درخواست فقط برای admin و با `public.is_admin()` فعال است.
- یک محافظ ساده برای درخواست‌های تکراری در بازه ۹۰ ثانیه وجود دارد.
- در آینده می‌توان Turnstile یا rate limit مبتنی بر IP را در edge/API اضافه کرد.

## تست

1. محصول به سبد اضافه کنید.
2. `/checkout` را باز کنید و فرم درخواست تماس را ثبت کنید.
3. مطمئن شوید cart پاک نشده است.
4. وارد `/admin/purchase-requests` شوید.
5. درخواست، اقلام، وضعیت‌ها، یادداشت و زمان پیگیری را بررسی کنید.
